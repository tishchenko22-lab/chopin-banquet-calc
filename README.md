# Документація проекту "Chopin Banquet Calculator"

## Опис проекту
Цей проект є калькулятором, який допомагає в розрахунках організації банкету, стильного та елегантного, на базі дизайну Фредеріка Шопена.

## Функції
- Розрахунок кількості порцій страв.
- Визначення вартості кожної страви.
- Генерація звіту про витрати на банкет.

## Використання
1. Клонувати репозиторій: `git clone https://github.com/tishchenko22-lab/chopin-banquet-calc.git`
2. Встановити залежності: `pip install -r requirements.txt`
3. Запустити програму: `python main.py`
